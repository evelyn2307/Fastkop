Fastkop - "Tempat Terbaik Untuk Membeli".

1. Sila baca keperluan sistem pada dokumen manualKEPERLUAN.pdf.

2. Sila baca cara pemasangan sistem pada dokumen manualPEMASANGAN.pdf.
	2.1 Pasang XAMPP dan mulakan Apache serta MySQL.
	2.2 Mulakan phpMyAdmin dan masukkan pangkalan data "dbkoop.sql".
	2.3 Copy folder "koop" dari folder "kod sumber" dan paste di C:\xampp\htdocs\
	2.3 Layari Fastkop di pelayar anda.

3. Sila baca dokumen manualPENGGUNA.pdf menggunakan perisian Adobe Acrobat Reader untuk mula menggunakan Fastkop.

4. Fail uploadPENGGUNAdataCSV.csv ialah fail contoh untuk muat naik data pengguna ke dalam pangkalan data melalui Fastkop.

5. Fail buku.jpg ialah gambar contoh untuk muat naik produk baru ke dalam pangkalan data melalui Fastkop.
